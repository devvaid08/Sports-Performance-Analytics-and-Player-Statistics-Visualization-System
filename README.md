# IPL Data Analytics Dashboard using Power BI

## Project Overview

This project is an interactive IPL Data Analytics Dashboard developed using Power BI. The dashboard provides season-wise analysis of teams, players, matches, venues, batting performance, and overall IPL statistics.

## Technologies Used

* Power BI Desktop
* Power Query
* DAX (Data Analysis Expressions)
* Python
* SQL Workbench
* Kaggle IPL Dataset

## Key Features

* Season-wise IPL Analysis (2008–2025)
* Champion and Runner-Up Tracking
* Orange Cap and Purple Cap Analytics
* Points Table Analysis
* Total Sixes Analysis
* Total Fours Analysis
* Match Analysis
* Team Analysis
* Venue Analysis
* Half-Century Analysis
* Century Analysis
* Dynamic KPI Cards
* Interactive Navigation Buttons
* Synchronized Season Slicers

## Dashboard Pages

1. Home Dashboard
2. Points Table Analysis
3. Total Sixes Analysis
4. Total Fours Analysis
5. Match Analysis
6. Team Analysis
7. Half-Centuries Analysis
8. Centuries Analysis
9. Venue Analysis

## Dataset Source

Kaggle IPL Dataset (2008–2025)
